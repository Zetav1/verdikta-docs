# Prerequisites

!!! warning "Content in Development"

## System Requirements

### Operating System
- Ubuntu 20.04 LTS or later
- macOS 11 (Big Sur) or later
- Windows 10/11 with WSL2

### Hardware Requirements
- **CPU**: 2+ cores recommended
- **RAM**: 6GB minimum, 8GB+ recommended
- **Storage**: 100GB+ free disk space
- **Network**: Stable broadband connection

### Software Dependencies
- Docker and Docker Compose
- Git
- Python 3.8+
- Node.js 16+

## API Keys Required

### AI Service Providers
- **OpenAI**: GPT-4 API access
- **Anthropic**: Claude API access
- **Local Models**: Optional Ollama setup

### Blockchain Services
- **RPC Provider**: Infura, Alchemy, or similar
- **IPFS Service**: Pinata, Infura IPFS, or self-hosted

### Wallet Setup
- Private key with sufficient funds
- Base Sepolia ETH for gas fees
- LINK tokens for oracle operations

## Network Configuration

### Firewall Settings
- Ensure required ports are open
- Configure NAT traversal if needed

### DNS Configuration
- Reliable DNS resolution
- Consider using 8.8.8.8 or 1.1.1.1
